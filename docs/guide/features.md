# 功能总览

OpenTrading 提供从数据获取、策略研究到模拟实盘的全流程量化工具链，所有功能均可在浏览器中操作。

---

## 1. 策略回测

<!-- 截图占位: 单股回测结果页 — 净值曲线 + 绩效指标 -->

支持对单只股票或 ETF 进行完整的历史回测，严格还原 A 股市场约束。

- **单股回测**：指定股票代码、日期区间和策略参数，一键运行并查看详细结果
- **A 股完整规则**：
  - T+1 持仓限制（当日买入次日方可卖出）
  - 涨跌停板限制（涨停无法买入，跌停无法卖出）
  - 印花税模拟（卖出按 0.1% 收取）
  - 整手约束（买入数量取 100 股整数倍）
- **Walk-forward 分析**：将历史区间滚动切分为 in-sample / out-of-sample 窗口，评估策略在陌生数据上的泛化能力
- **显著性检验**：基于 Block Bootstrap 方法检验回测结果是否显著优于随机基准，输出 p 值和置信区间

---

## 2. 因子研究

<!-- 截图占位: 因子研究面板 — IC 时序图 + 分组收益柱状图 -->

提供完整的截面因子评估体系，支持内置因子和用户自定义因子。

- **IC / RankIC / ICIR**：计算因子与下期收益的信息系数，评估因子预测能力
- **IC 衰减分析**：观察因子信号在未来多个周期的持续有效性
- **分组收益**：将股票按因子值分组，统计各组历史平均收益，验证单调性
- **相关性矩阵**：多因子之间的 Pearson 和 Spearman 相关性热力图，辅助因子去重
- **中性化**：支持对市值、行业等风险因子进行中性化处理，提取纯净因子信号
- **多因子合成**：等权、IC 加权、IR 加权等方式合成复合因子

---

## 3. 组合回测

<!-- 截图占位: 组合回测结果页 — 净值曲线 + Brinson 归因图表 -->

支持多标的组合的历史回测，提供多种优化器和风控机制。

- **内置组合策略**：
  - 行业/市值轮动策略
  - 多因子选股策略
  - ETF MACD 趋势跟踪策略
- **组合优化器**：
  - 均值方差优化（Mean-Variance）
  - 风险平价（Risk Parity）
  - 最小方差优化
  - Ledoit-Wolf 收缩协方差估计，改善矩阵病态问题
- **风控机制**：
  - 最大回撤熔断（触发后暂停调仓）
  - 单标的仓位上限 / 下限约束
  - 换手率限制（控制交易成本）
  - 预交易风控引擎
- **Brinson 归因**：将组合超额收益拆分为资产配置效应、选股效应和交互效应
- **批量参数搜索**：网格搜索或随机搜索策略参数，配合 Walk-forward 防止过拟合

## 4. MLAlpha

<!-- 截图占位: MLAlpha 配置与诊断面板 -->

将机器学习引入因子收益预测，内置完整的过拟合诊断体系。

- **支持的估计器**（9 种）：
  - 线性模型：`Ridge`、`Lasso`、`LinearRegression`、`ElasticNet`
  - 树模型：`DecisionTreeRegressor`、`RandomForestRegressor`、`GradientBoostingRegressor`
  - 梯度提升：`LGBMRegressor`、`XGBRegressor`
- **Walk-forward 训练**：滚动训练窗口，使用 Purge / Embargo 方法消除训练集与测试集之间的信息泄漏
- **过拟合诊断**：
  - 特征重要性稳定性分析（跨窗口一致性）
  - IS / OOS IC 衰减对比
  - 换手率监控（过高换手率往往是过拟合信号）
  - 综合过拟合诊断结论（pass / warn / fail）

## 5. AI 助手

<!-- 截图占位: AI 助手对话界面 — 策略生成示例 -->

基于大语言模型的对话式量化开发助手。

- **对话式策略开发**：用自然语言描述策略逻辑，AI 自动生成符合框架规范的 Python 代码
- **代码生成 + 回测 + 评估**：AI 可在同一对话中完成策略编写、触发回测并解读结果
- **自主研究模式**：设定研究目标后，AI Agent 端到端完成因子挖掘、策略开发和验证，中间步骤自动执行
- **沙箱安全**：
  - 阻止危险模块导入（如 `os`、`subprocess`）
  - 禁止访问系统敏感路径
  - 代码保存前自动执行前视偏差、NaN/Inf、权重合法性等守卫检查
- **支持模型**：DeepSeek（推荐）、OpenAI 及所有 OpenAI 兼容接口

## 6. 模拟实盘

<!-- 截图占位: 模拟实盘监控仪表盘 — 持仓状态 + 调度日志 -->

从策略研究到每日自动调仓的完整部署管道，面向严肃的量化交易者。

- **部署门控（DeployGate）**：策略部署前必须通过比回测更严格的合规性检查
- **每日调仓调度器**：
  - 幂等设计，重启后自动从上次状态恢复
  - 每个部署独立加锁，防止并发冲突
  - 支持可选后台自动 tick（环境变量开关）
- **OMS 事件溯源**：
  - 所有委托、成交、撤单事件持久化存储
  - 支持幂等重放（重复事件自动去重）
  - 策略状态跨重启持久化
- **监控仪表盘**：持仓实时状态、调度历史日志、风控触发记录、Webhook 告警
- **预交易风控引擎**：委托前校验仓位上限、资金充足性、资本策略阶段约束
- **QMT 券商对接**：
  - 原生对接迅投 QMT 交易系统
  - 支持影子账户验证和小白名单实盘提交
  - 四维对账闭环（账户 + 委托 + 持仓 + 成交）
  - 回调驱动的生命周期闭环

## 7. 数据层

<!-- 截图占位: 数据管理页 — 缓存状态与数据源配置 -->

多源数据接入，本地缓存优先，支持离线使用。

- **Tushare**：A 股日线行情、基本面数据（主数据源）
- **AKShare**：ETF 净值、部分备用行情数据
- **本地 Parquet 缓存**：数据下载后存储为 Parquet 文件，后续读取速度极快，不消耗 API 积分
- **DuckDB**：回测结果、策略配置、部署记录等元数据存储
- **数据哈希**：组合回测记录 `_data_hash` 用于比较不同运行之间的数据一致性（仅警告，不阻断运行）
