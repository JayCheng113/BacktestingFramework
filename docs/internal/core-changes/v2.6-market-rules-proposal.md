# Core-Change Proposal: MarketRules (V2.6)

> **状态**: 提案
> **影响范围**: engine.py (1 行), matcher.py (0 行), 新文件 market_rules.py
> **向后兼容**: 是 — 现有 Matcher 不受影响

## 问题

当前回测不模拟 A 股市场规则：
- **T+1**: 买入当天不可卖出（当日买入次日才能卖）
- **涨停不可买**: 价格涨幅达 10%（或 20% 创业板/科创板），无法买入
- **跌停不可卖**: 价格跌幅达 10%（或 20%），无法卖出
- **整手交易**: A 股最小 100 股，必须 100 的倍数
- **停牌**: 无成交，不可交易（已通过 NaN 价格防护解决）

这导致回测结果偏乐观 — 真实交易中这些规则会减少交易次数、降低收益。

## 设计方案

### 核心原则：最小改动

Matcher ABC **不改**。Engine 加 **1 行**。所有市场规则逻辑放在**新文件**。

### 技术方案

**1. Engine 改动（1 行）**

```python
# ez/backtest/engine.py _simulate() 循环内，exec_price 之后加：
for i in range(1, n):
    ...
    exec_price = open_prices[i]
    # V2.6: notify matcher of bar context (backward compatible)

```

现有 SimpleMatcher / SlippageMatcher 没有 `on_bar` → `hasattr` 返回 False → 跳过。零影响。

**2. MarketRulesMatcher（新文件，装饰器模式）**

```python
# ez/core/market_rules.py
class MarketRulesMatcher(Matcher):
    """Wraps an inner Matcher with A-share market rules."""

    def __init__(self, inner: Matcher, t_plus_1: bool = True,
                 price_limit_pct: float = 0.1, lot_size: int = 100):
        self._inner = inner
        self._t1 = t_plus_1
        self._limit = price_limit_pct
        self._lot = lot_size
        self._buy_bar = -1
        self._bar = 0
        self._prev_close = 0.0

    def on_bar(self, bar_index: int, prev_close: float):
        self._bar = bar_index
        self._prev_close = prev_close

    def fill_buy(self, price, amount):
        # 涨停不可买
        if self._prev_close > 0 and price >= self._prev_close * (1 + self._limit - 1e-6):
            return zero_fill(price)
        fill = self._inner.fill_buy(price, amount)
        # 整手交易
        if self._lot > 0 and fill.shares > 0:
            lots = int(fill.shares // self._lot)
            if lots == 0: return zero_fill(price)
            actual = lots * self._lot
            # 退还多余现金...
        if fill.shares > 0:
            self._buy_bar = self._bar
        return fill

    def fill_sell(self, price, shares):
        # T+1
        if self._t1 and self._buy_bar == self._bar:
            return zero_fill(price)
        # 跌停不可卖
        if self._prev_close > 0 and price <= self._prev_close * (1 - self._limit + 1e-6):
            return zero_fill(price)
        return self._inner.fill_sell(price, shares)
```

**3. 向后兼容保证**

```python
# 不带 market_rules:
engine = VectorizedBacktestEngine(matcher=SimpleMatcher())
# → hasattr(SimpleMatcher, 'on_bar') = False → 跳过 → 行为不变

# 带 market_rules:
inner = SlippageMatcher(slippage_rate=0.001)
matcher = MarketRulesMatcher(inner, t_plus_1=True, lot_size=100)
engine = VectorizedBacktestEngine(matcher=matcher)
# → hasattr(MarketRulesMatcher, 'on_bar') = True → 每 bar 通知 → 规则生效
```

## 验证方案

| 测试 | 断言 |
|------|------|
| T+1: 买入当 bar 卖出信号被阻止 | shares 不变，无 sell fill |
| T+1: 买入次 bar 卖出成功 | fill.shares > 0 |
| 涨停: 涨幅=10% 时 fill_buy 返回 0 | zero_fill |
| 跌停: 跌幅=10% 时 fill_sell 返回 0 | zero_fill |
| 整手: 123 股→100 股 | fill.shares = 100 |
| 整手: 不足 100 股→跳过 | fill.shares = 0 |
| 无规则: SimpleMatcher 行为不变 | 全量 invariant tests pass |
| 回归: 779 tests 不回归 | 全 pass |

## 风险评估

| 风险 | 缓解 |
|------|------|
| engine.py 改动引入回归 | hasattr 检查 + 全量 invariant tests |
| 整手交易退款计算错误 | 退还多余现金 = amount - (actual_shares * price + commission) |
| T+1 跨 bar 状态泄漏 | `_buy_bar` 只在 fill_buy 时更新，on_bar 只更新 bar_index |
| 涨跌停阈值误判 | 用 epsilon 容差 (1e-6) |
